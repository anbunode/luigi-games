# SkrepayShop Platform

Lobby y login oficial del ecosistema SaaS multi-tenant.

- **Producción:** https://strong-cascaron-2e0511.netlify.app
- **Dominio futuro:** https://skrepay.com
- **Panel Medusa:** https://luigi-games-api1.onrender.com/app

## Rutas

| Ruta | Descripción |
|------|-------------|
| `/` | Lobby con video hero |
| `/login` | Inicio de sesión oficial |
| `/api/auth/login` | Auth contra API Render |

## Desarrollo

```bash
cd skrepayshop-platform
npm install
npm run dev
```

http://localhost:3001
